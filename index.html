Dashboard Overview Screen

Dashboard Features

Interactive Filtering

Filter data by region, gender, age group, and skill type. Examine specific skills across different demographics.

Regional Analysis

Compare skills across different regions. Identify regional priorities and specific training needs.

Gender Comparison

Analyze how training needs differ between genders. Identify gender-specific priorities.

Demographic Insights

Visualize the distribution of trainees by region, gender, and age group with interactive charts.

Trainee Database

Access a detailed database of all trainees with filtering and sorting capabilities.

Training Needs Analysis

Identify the most requested skills overall and by specific demographic groups.

How to Run the Full Dashboard

  1. Clone the repository: 
    git clone https://github.com/Rojlap107/regional-training-dashboard.git
cd regional-training-dashboard
  2. Run the dashboard script: 
    ./run_dashboard.sh
  3. Open your web browser and go to: http://127.0.0.1:8050/
